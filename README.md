# Download Prefixer

İndirilen dosyaların başına domain bazlı prefix ekleyen Chrome uzantısı.

## Kurulum

1. Chrome'da `chrome://extensions/` adresine gidin
2. **Geliştirici modu**'nu etkinleştirin
3. **Paketlenmemiş uzantı yükle** → proje klasörünü seçin

## Kullanım

1. Prefix eklemek istediğiniz siteye gidin
2. Uzantı ikonuna tıklayın → **+ Prefix Ekle**
3. Domain otomatik doldurulur, prefix girin ve kaydedin

**Örnek:** `github.com` + `gh_` → İndirilen dosya: `gh_dosya.zip`
